R1M3

Controls:

  • Hold "n" and press a number key to change the value of n (two digit values of n yet to come)
  • Type k numbers and press enter to create a rim
  • Pressing tab or shift+tab will cycle through which rims are selected (they are "bold" when selected)
  • Arrow keys: move selected rim
  • c/x: cycle selected rim
  • f: vertically flip selected rim
  • d: duplicate selected rim
  • g: toggle grid points
  • +/-: Increase or decrease grid size
  • l: toggle labels
